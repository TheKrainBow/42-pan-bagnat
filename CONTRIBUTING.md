# Contributing to Pan Bagnat

Thank you for your interest in contributing to Pan Bagnat.

Pan Bagnat is a community-driven project with centralized governance to ensure long-term
consistency, security, and sustainability.

────────────────────────────────────────────────────────────
## Contribution Rules
────────────────────────────────────────────────────────────

- All contributions MUST be submitted via Pull Request
- Direct commits to main branches are not allowed
- Pull Requests are reviewed and approved solely by the project owner
- Approval is at the discretion of the maintainers

────────────────────────────────────────────────────────────
## Legal Notice
────────────────────────────────────────────────────────────

By submitting a Pull Request, you explicitly agree that:
- Your contribution is provided under the Contributor License Agreement (CLA)
- Ownership of the contribution is transferred to the project owner
- You waive any claim of ownership or control over the contributed code

If you do not agree with these terms, please do not submit contributions.

────────────────────────────────────────────────────────────
## Scope
────────────────────────────────────────────────────────────

The project is public and source-available but NOT open-source under OSI definitions.
Commercial use is strictly prohibited without explicit permission.

────────────────────────────────────────────────────────────
## Code Quality
────────────────────────────────────────────────────────────

- Keep changes focused and minimal
- Follow existing architecture and patterns
- Security and maintainability have priority over features

────────────────────────────────────────────────────────────
## Communication
────────────────────────────────────────────────────────────

Be respectful and constructive. Maintainers may close discussions or PRs that do not
align with the project direction.
